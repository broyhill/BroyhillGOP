## DATABASE — LINKING INFLUENCERS TO SEGMENTS

```sql
-- influencer_segment_map — each influencer tagged to their segments
CREATE TABLE IF NOT EXISTS public.influencer_segment_map (
    influencer_id       integer REFERENCES public.influencer_registry(id),
    segment_tag         text,   -- 'farmers' | 'doctors' | 'hunters' | etc.
    segment_rank        integer,  -- their rank within this segment's leaderboard
    segment_score       numeric,  -- their score WITHIN this segment only
    is_segment_leader   boolean DEFAULT false,  -- top 10% in segment
    PRIMARY KEY (influencer_id, segment_tag)
);

-- Per-segment leaderboard query
SELECT
    ir.full_name,
    ism.segment_rank,
    ism.segment_score,
    ir.total_dollars_referred,
    ir.total_donors_referred,
    ir.influencer_tier,
    ir.relationship_status,
    ir.last_contact_date
FROM public.influencer_segment_map ism
JOIN public.influencer_registry ir ON ir.id = ism.influencer_id
WHERE ism.segment_tag = 'farmers'   -- swap for any segment
ORDER BY ism.segment_rank ASC;
```

## CROSS-SEGMENT INFLUENCERS — The Most Valuable

Some people have credibility in MULTIPLE segments simultaneously:

| Person Type | Segments They Bridge | Multiplied Value |
|------------|---------------------|-----------------|
| Doctor who hunts | Healthcare + Hunters | Rare — both worlds trust them |
| Farm Bureau board member who's also a deacon | Farmers + Faith | Rural NC gold |
| Retired sheriff who farms | Law Enforcement + Farmers | Rural county kingmaker |
| Soccer mom who's also a nurse | Soccer Moms + Nurses | Suburban + healthcare |
| Dairy farmer who's a veteran | Dairy + Veterans | Deep loyalty both ways |
| Realtor who's active in church | Real Estate + Faith | Suburban community |

```sql
-- Find cross-segment influencers (in 3+ segments)
SELECT ir.full_name, ir.influencer_score, COUNT(ism.segment_tag) as segment_count,
    ARRAY_AGG(ism.segment_tag) as segments
FROM public.influencer_registry ir
JOIN public.influencer_segment_map ism ON ism.influencer_id = ir.id
GROUP BY ir.id, ir.full_name, ir.influencer_score
HAVING COUNT(ism.segment_tag) >= 3
ORDER BY ir.influencer_score DESC;
-- These people are RARE and EXTREMELY VALUABLE — treat as Platinum regardless of score
```
