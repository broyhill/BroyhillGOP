### 🎣 COASTAL COMMERCIAL FISHING
**Communities:** Beaufort, Morehead City, Harkers Island, Swansboro, Oriental, Wanchese, Engelhard
**Permanent issues:**
- Commercial shrimp trawling vs recreational fishing (biggest fight)
- Gill net restrictions — NCWRC season rules
- Shrimp trawl ban bills (HB442 style) — existential threat to commercial fleet
- Crab/oyster aquaculture licensing
- Bycatch regulations — sea turtle, sturgeon
- NOAA federal catch limits
- Commercial fishing license transferability
- Ice house / seafood processing industry survival
**Special interests engaged:**
- NC Fisheries Association (commercial) → N/D lean — opposes restrictions
- CCA NC (recreational) → R lean — pushes trawl restrictions
- Congressional Sportsmen's Foundation → R lean
- NOAA Fisheries — federal regulator
- NC Coastal Federation — environmental, sometimes opposes both sides
**community_identity_tag:** `coastal_commercial_fishing`

## DATABASE IMPLEMENTATION

```sql
-- community_identity_types lookup table
CREATE TABLE IF NOT EXISTS public.community_identity_types (
    tag             text PRIMARY KEY,
    label           text NOT NULL,
    description     text,
    typical_partisan_lean char(1) DEFAULT 'N'
);

-- community_profiles table
CREATE TABLE IF NOT EXISTS public.community_profiles (
    id              SERIAL PRIMARY KEY,
    community_name  text NOT NULL,
    county          text[],
    region          text,
    identity_tags   text[],   -- array of community_identity_type tags
    population      integer,
    notes           text,
    created_at      timestamptz DEFAULT NOW()
);

-- Link communities to candidates/districts
CREATE TABLE IF NOT EXISTS public.community_candidate_map (
    community_id    integer REFERENCES public.community_profiles(id),
    candidate_id    bigint REFERENCES public.candidates(id),
    relevance       text    -- 'primary_community' | 'secondary_community'
);
```
